# 签到系统使用文档

## 1. 功能简介

签到系统是博客平台的用户激励功能，通过每日签到获取积分奖励，连续签到可获得额外加成。

### 1.1 核心功能

| 功能 | 说明 |
|------|------|
| 每日签到 | 每天可签到一次，获取基础积分 |
| 连续签到奖励 | 连续签到达到特定天数，获得额外积分 |
| 早起奖励 | 早上 6:00-8:00 签到，额外获得积分 |
| 签到日历 | 可视化展示签到记录 |
| 签到统计 | 展示累计签到天数、最长连续天数等 |

---

## 2. 用户操作指南

### 2.1 签到入口

**位置**：个人中心 → 签到面板

```
首页 → 个人中心 → 签到
```

### 2.2 签到操作

1. **进入签到页面**
   - 登录后点击底部导航「我的」
   - 在个人中心顶部找到签到面板

2. **点击签到按钮**
   - 点击「签到」按钮完成签到
   - 系统自动计算积分并展示结果

3. **查看签到结果**
   - 弹窗显示获得的积分
   - 显示连续签到天数
   - 特殊奖励（早起、连续达标）会有额外提示

### 2.3 签到日历

在签到面板下方，可以查看当月签到日历：

- ✅ 已签到的日期显示对勾
- 🔥 当天日期高亮显示
- 点击日历可查看当天获得的积分

## 3. 积分规则

### 3.1 基础积分

| 签到类型 | 积分 | 说明 |
|----------|------|------|
| 每日签到 | +5 | 基础签到奖励 |
| 早起签到 | +5 | 6:00-8:00 签到额外奖励 |

### 3.2 连续签到奖励

| 连续天数 | 额外积分 | 说明 |
|----------|----------|------|
| 7 天 | +20 | 一周坚持 |
| 14 天 | +50 | 两周不间断 |
| 30 天 | +100 | 月度达人 |
| 60 天 | +200 | 双月坚持 |
| 100 天 | +500 | 百日成就 |
| 180 天 | +1000 | 半年守望 |
| 365 天 | +3000 | 年度传奇 |

**示例计算**：

```
连续签到第7天：
- 基础积分：5
- 连续7天奖励：20
- 早起奖励：5（如果在6-8点签到）
- 总计：30 积分
```

### 3.3 断签说明

- 如果某天未签到，连续天数将重置为 1
- 累计签到天数不受影响
- 可以从头开始积累连续天数

## 5. 常见问题

### Q1: 每天什么时间可以签到？

**A**: 每天 00:00 后即可签到，签到记录以自然日为准。

### Q2: 忘记签到了怎么办？

**A**: 
- 可以使用补签卡补签（消耗积分购买）
- 连续天数会重置，需要重新积累
- 建议开启签到提醒通知

### Q3: 早起奖励的时间范围是？

**A**: 早上 6:00-8:00 签到可获得额外 5 积分奖励。

### Q4: 积分有上限吗？

**A**: 签到积分没有上限，但每日签到只能获得一次积分。

### Q5: 换设备签到会怎样？

**A**: 签到记录与账号绑定，换设备不影响签到数据和连续天数。

## 6. 小技巧

### 🎯 快速积累积分

1. **早起签到** - 每天早上签到可获得额外 5 积分
2. **保持连续** - 连续签到的奖励远高于断断续续
3. **坚持30天** - 30天连续签到可获得 100 积分大奖

### 📅 签到提醒

建议开启浏览器的推送通知，或绑定微信提醒，避免遗忘签到。

### 🏆 成就联动

签到还可以解锁成就徽章：
- 连续签到 7 天 → 解锁「坚持一周」成就
- 连续签到 30 天 → 解锁「月度达人」成就
- 连续签到 100 天 → 解锁「百日坚持」成就
- 连续签到 365 天 → 解锁「年度守望」成就
