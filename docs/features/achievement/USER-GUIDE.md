# 成就徽章系统使用文档

## 1. 功能简介

成就徽章系统是博客平台的游戏化激励功能，通过完成各种任务解锁成就徽章，获得积分奖励。

### 1.1 成就分类

| 分类 | 说明 | 示例 |
|------|------|------|
| 📝 内容成就 | 与创作相关的成就 | 发布文章、获得点赞 |
| 👥 社交成就 | 与互动相关的成就 | 评论、粉丝、关注 |
| 🎯 活动成就 | 参与活动的成就 | 签到、连续签到 |
| ⭐ 特殊成就 | 特殊条件的成就 | 早起签到、周年纪念 |

### 1.2 成就等级

| 等级 | 标识 | 稀有度 | 示例 |
|------|------|--------|------|
| 普通 | 无标识 | 常见 | 初出茅庐、初次交流 |
| 稀有 | 绿色标识 | 较少见 | 笔耕不辍、人气之星 |
| 史诗 | 紫色标识 | 稀有 | 文坛巨匠、意见领袖 |
| 传说 | 金色标识 | 极稀有 | 传奇作者、年度守望 |

## 4. 成就解锁流程

### 4.1 自动解锁

大多数成就会在完成条件后自动解锁：

1. 完成对应操作（发布文章、评论、签到等）
2. 系统自动检查成就条件
3. 满足条件时自动解锁
4. 弹出解锁通知
5. 积分自动发放到账户

### 4.2 解锁通知

成就解锁时会收到：
- 站内通知消息
- 成就解锁弹窗
- 积分到账提醒

## 7. 常见问题

### Q1: 成就可以重复解锁吗？

**A**: 每个成就只能解锁一次，但可以同时完成多个成就。例如发布第 10 篇文章时，如果还没解锁「初出茅庐」，会同时解锁两个成就。

### Q2: 删除文章会影响成就吗？

**A**: 不会。成就以累计数据为准，已解锁的成就不会被撤销。

### Q3: 成就进度如何计算？

**A**: 进度基于累计数据：
- 发布文章数：所有已发布的文章
- 点赞数：所有文章获得的点赞总数
- 粉丝数：当前粉丝数量

### Q4: 特殊成就如何触发？

**A**: 特殊成就需要特定条件：
- 「早起的鸟儿」：在早上 6-8 点签到时自动触发
- 「完美档案」：完善昵称、头像、简介等所有个人资料后触发
- 「一周年纪念」：注册满一年当天触发

### Q5: 为什么完成条件但没有解锁？

**A**: 可能的原因：
1. 数据延迟，稍后刷新页面
2. 条件理解有误（如连续签到和累计签到的区别）
3. 系统缓存，尝试刷新页面或重新登录

## 8. 小技巧

### 🎯 快速解锁成就

1. **从简单开始**
   - 先解锁「初出茅庐」「初次交流」等简单成就
   - 积少成多，积累积分

2. **连续签到**
   - 保持签到习惯，同时解锁多个签到成就
   - 还可以触发「早起的鸟儿」特殊成就

3. **积极互动**
   - 多评论、多关注
   - 高质量内容吸引粉丝

4. **完善资料**
   - 一次性完善所有资料，解锁「完美档案」
   - 还可以解锁「初次编辑」

### 📊 成就规划

建议解锁顺序：
1. ✅ 首次操作类（发布文章、评论、签到）
2. ✅ 累计数量类（10 篇文章、10 条评论）
3. ✅ 社交互动类（粉丝、关注）
4. ✅ 长期积累类（100 篇文章、连续签到）
